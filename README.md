# Popsicle Stick Bridge – University Competition 2026

This repository contains the **SolidWorks design files, photos, and report** for our popsicle stick bridge built for the 2026 University Engineering Competition.

## Project Overview
- **Objective:** Design and construct a bridge using only popsicle sticks and glue.
- **Materials:** Popsicle sticks, wood glue.
- **Design Tools:** SolidWorks for modeling, testing, and simulations.

## Contents
- **SolidWorks/** – Assembly and part files with drawings.
- **Photos/** – Pictures of the actual bridge.
- **Report/** – Detailed design and testing report.
